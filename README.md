# 使用方式

1. 安裝好nvm後clone本專案
2. 透過終端機輸入
npm install
3. 安裝完npm後輸入
npm run dev
4. 若看見以下訊息表示運作成功
Listening on http://*******:******
※如要停止請輸入兩次 ctrl + C
